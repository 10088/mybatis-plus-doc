# Mybatis-Plus Doc Website
This repo is the site of `Mybatis-Plus`, the generated files at gh-pages branch.

# Improve this doc
```bash
# clone this repo
git clone -b master https://github.com/baomidou/mybatis-plus-doc.git

# cd into project
cd mybatis-plus-doc/

# install
npm install

# generate
hexo generate & gulp

# run as serve
hexo serve
```
