# Mybatis-Plus Doc Website
This repo is the site of `Mybatis-Plus`, the generated files at gh-pages branch.

# Improve this doc
```bash
# clone this repo
git clone https://github.com/baomidou/mybatis-plus-doc.git

# cd into project
cd mybatis-plus-doc/

# install gitbook-cli
npm install gitbook-cli -g

# init gitbook
gitbook install

# run as serve
gitbook serve
```
