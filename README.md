# Mybatis-Plus

Mybatis-Plus 是 `Mybatis` 的增强工具包，为简化开发工作、提高生产率而生。

+ 我们的愿景是成为`Mybatis`最好的搭档，就像魂斗罗中的2P愿意和1P一起共面难题一样。
+ Mybatis-Plus只做增强不做改变。
