- 润物无声
  - 只做增强不做改变，引入它不会对现有工程产生影响，如丝般顺滑。
- 效率至上
  - 只需简单配置，即可快速进行 CRUD 操作，从而节省大量时间。
- 丰富功能
  - 热加载、代码生成、分页、性能分析等功能一应俱全。
