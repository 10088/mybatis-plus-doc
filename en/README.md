## Help & Improve

> Welcome everybody commit issue & pull-request！

``` bash
# clone this repo
git clone https://github.com/baomidou/mybatis-plus-cn-doc.git

# cd into project
cd mybatis-plus-cn-doc/

# install gitbook-cli
npm install gitbook-cli -g

# init gitbook
gitbook install

# run as serve
gitbook serve

```
