# Help & Improve

> Welcome everybody commit issue & pull-request！

```bash
# clone this repo
git clone https://github.com/baomidou/mybatis-plus-doc.git

# cd into project
cd mybatis-plus-doc/

# install gitbook-cli
npm install gitbook-cli -g

# init gitbook
gitbook install

# run as serve
gitbook serve
```
